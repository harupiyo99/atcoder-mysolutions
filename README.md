# atcoder-mysolutions

AtCoderで解いた問題の自分の解答コードを保存するリポジトリです。

## 記録
- **目標:** 毎週のABC（AtCoder Beginner Contest）に参加する。4月までにはレート400を超える。
- **使用言語:** C++
- **現在のランク:** 灰色(107)
- **解く問題:** ほとんどABCの問題。AtCoder ProblemsのRecomendation・Trainingや参加した回の問題中心。

## フォルダ構成
- `ABC/`: AtCoder Beginner Contest
